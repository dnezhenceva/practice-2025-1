О проекте Участники Журнал Ресурсы

Аннотация

В рамках проектной практики первого курса был разработан информационный сайт, демонстрирующий основы верстки, использования Git и создания статических веб-страниц.

Техническая реализация

Центральным этапом проекта стала техническая реализация (вариативная часть проекта) — создание Telegram-бота.

Telegram Horoscope Bot — это Telegram-бот, предназначенный для получения гороскопов по знаку зодиака. Пользователи могут выбрать свой знак, указать интересующую дату и получить персональный гороскоп с переводом на русский язык.

Бот использует внешний API для получения актуальных астрологических данных и переводит тексты с английского на русский через MyMemory API.

Основные функции

  • /start — запуск и приветствие
  • /помощь или Помощь — справка по функциям бота
  • /гороскоп или Гороскоп — начать выбор знака зодиака и даты

Логика диалога

  1. Пользователь нажимает «Гороскоп» или отправляет команду /гороскоп.
  2. Бот предлагает выбрать знак зодиака.
  3. Затем предлагает указать день: сегодня, завтра, вчера или вручную дату в формате ГГГГ-ММ-ДД.
